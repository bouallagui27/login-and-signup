# 🚀 My First Full-Stack Project: Login & Signup System

Welcome to my first web development project! This is a secure **User Authentication System** (Login & Register) built to learn the fundamentals of Full-Stack development .

---

## 🛠️ Languages & Tools Used

### Frontend (Client)
- **Language**: JavaScript (React.js)
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM

### Backend (Server)
- **Language**: Node.js & Express.js
- **Database**: MariaDB (via XAMPP)
- **ORM**: Sequelize
- **Security**: Bcrypt for password hashing

---

## 💡 What I Learned
In this project, I successfully implemented:
1. **Frontend-Backend Connection**: Sending data from React to a Node.js server using `fetch`.
2. **Database Management**: Creating and managing user tables in MariaDB.
3. **Cybersecurity Basics**: Protecting user data by hashing passwords with `bcrypt` before saving them to the database.
4. **Asynchronous Logic**: Using `async/await` to handle database operations smoothly.

---

## 🏗️ Project Structure
- `/client`: Contains the React application (UI).
- `/backend`: Contains the Node.js server and API logic.

---

## ⚙️ How to Run
1. Start **XAMPP** (Apache & MySQL).
2. Navigate to `/backend` and run `node serveur.js`.
3. Navigate to `/client` and run `npm run dev`.
